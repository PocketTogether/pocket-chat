主要主题颜色的scss
`E:\Project\pocket-chat\vue3\src\assets\styles\element-plus.scss`
```scss
@use "sass:color";

@function color-lighten($color, $step) {
  @return color.mix(#ffffff, $color, $step * 10%);
}

@function color-darken($color, $step) {
  @return color.mix(#181818, $color, $step * 10%);
}

// 配置 element-plus 颜色
$types: (
  primary: #7899CC,
  success: #799978,
  info: #5A7583,
  warning: #BB9854,
  danger: #AB4378,
  error: #AB4378
);

$light-steps: 1, 2, 3, 4, 5, 6, 7, 8, 9;
$dark-steps: 1, 2, 3, 4, 5, 6, 7, 8, 9;

:root:root {
  @each $type, $color in $types {
    --el-color-#{$type}: #{$color};

    @each $step in $light-steps {
      --el-color-#{$type}-light-#{$step}: #{color-lighten($color, $step)};
    }

    @each $step in $dark-steps {
      --el-color-#{$type}-dark-#{$step}: #{color-darken($color, $step)};
    }
  }
}

:root:root.dark {
```

以上需要注意的是 xxx-light-1 或 xxx-dark-1，先拿xxx-light-1说事吧
xxx-light-1其并不是绝对的更白的的意思，绝不能误解为更白的的意思，因为其在明色或暗色模式下会是不同的值
比如在明色时，light指的是更白，但在暗色时，light指的是更黑，一定要搞清楚
比如在明色时，dark指的是更黑，但在暗色时，dark指的是更白，一定要搞清楚


主要背景、文字这类的颜色的css变量，
其是在 `E:\Project\pocket-chat\vue3\src\assets\styles\color.scss` 里写的
其在明色或暗色模式下会是不同的值，以达到效果
但你并不需要在意细节，你只要知道有以下css量即可
```
  --color-background: xxxx;
  --color-background-soft: xxxx;
  --color-background-mute: xxxx;
  --color-text: xxxx;
  --color-text-soft: xxxx;

color-background 顾名思义即为背景色
color-background-soft 和背景色有一些差别，常用于一些内容块的背景色
color-background-mute 和背景色的差别程度更大一些
color-text 顾名思义即为 文字色
color-text-soft 也常用于文字色，更淡一点而已
```
