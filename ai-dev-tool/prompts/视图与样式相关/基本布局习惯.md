pocket-chat，没有全局性的侧栏、顶栏或底栏。
一般顶栏或底栏，都在单独的某列中


一般习惯为单列布局或双列布局

单列布局可参考
`E:\Project\pocket-chat\vue3\src\views\file-info\FileInfoPage.vue`
```vue
<template>
  <div>
    <!-- 边距控制 -->
    <div class="mx-[8px]">
      <div
        class="mx-auto"
        :style="{
          maxWidth: '500px',
        }"
      >
        <div class="relative">
          <!-- 顶栏 -->
          <div class="sticky top-0 z-[1] flow-root">
            <!-- 文件详情页顶栏 -->
          </div>
          <!-- 内容 -->
          <!-- 加载状态 -->
          <!-- 文件不存在 -->
        </div>
      </div>
    </div>
  </div>
</template>
```

另一种较宽的单列布局可参考
`E:\Project\pocket-chat\vue3\src\views\chat-home\ChatHome.vue`
```vue
<template>
  <div
    :class="{
      /* 窗口宽度大于600时聊天栏边距宽度较大，小于则聊天栏边距宽度较小 */
      'mx-[40px]': showChatWidthLargerTrueWidthSmallerFalse,
      'mx-[8px]': !showChatWidthLargerTrueWidthSmallerFalse,
    }"
  >
    <div
      class="mx-auto max-w-[768px]"
    >
      <!-- 这是聊天列内容。。。 -->
    </div>
  </div>
</template>
```


双列布局可参考
`E:\Project\pocket-chat\vue3\src\views\file\FileSelectPage.vue`
```vue
<script setup lang="ts">
import {
  ContainerBar,
  ContainerCol2,
} from '@/components'
// ...
</script>
<template>
  <div>
    <!-- 大屏 双列 -->
    <div v-if="windowSize.width.value >= 768" class="">
      <ContainerCol2
        col1Position="right"
        :col2StyleValue="{
          width: (() => {
            // 大于1024时左栏宽度较大，小于则左栏宽度较小
            if (windowSize.width.value >= 1024) {
              return `500px`
            }
            return `400px`
          })(),
        }"
        col1Twcss="flex-1 overflow-hidden"
      >
        <!-- 左侧 | 顶栏 底栏 操作面板 -->
        <template #col2>
          <div class="ml-6 mr-4">
            <ContainerBar :defaultBarHeight="60">
              <template #default>
                <div class="relative mb-4">
                  <div class="sticky top-0 z-[1] flow-root">
                    <!-- 文件页顶栏 -->
                  </div>
                  <!-- 操作面板 -->
                  <!-- 上传列表 -->
                  <!-- 文件预览 -->
                </div>
              </template>
              <template #bar>
                <div class="flow-root">
                  <!-- 文件页底栏 -->
                </div>
              </template>
            </ContainerBar>
          </div>
        </template>
        <!-- 右侧 | 文件列表 -->
        <template #col1>
          <div class="my-4 ml-2 mr-6">
            <!-- 文件列表 -->
          </div>
        </template>
      </ContainerCol2>
    </div>
    <!-- 小屏 单列 -->
    <div v-else>
      <!-- 顶栏 底栏 操作面板 文件列表 -->
      <div class="mx-[8px]">
        <div
          class="mx-auto"
          :style="{
            maxWidth: '500px',
          }"
        >
          <ContainerBar :defaultBarHeight="52">
            <template #default>
              <div class="relative mb-4">
                <div class="sticky top-0 z-[1] flow-root">
                  <!-- 文件页顶栏 -->
                </div>
                <!-- 操作面板 -->
                <!-- 上传列表 -->
                <!-- 文件列表 -->
              </div>
            </template>
            <template #bar>
              <!-- 文件页底栏 -->
            </template>
          </ContainerBar>
        </div>
      </div>
    </div>
  </div>
</template>
```
双列布局时一般都要考虑大小屏响应式
