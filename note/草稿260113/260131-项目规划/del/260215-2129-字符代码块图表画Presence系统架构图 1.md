# ⭐ 1. Mermaid（最推荐）

你绝对会喜欢它，因为：

- GitHub 原生支持
- Markdown 原生支持
- 可以导出为 ASCII、SVG、PNG
- 语法极其简单
- 适合画流程图、时序图、状态图、ER 图

示例：

```
flowchart TD
  A[Frontend] --> B[userPresencesStatus]
  A --> C[userNotViewingMarks]
  B --> D[Presence Merge Logic]
  C --> D
```

你可以用 Mermaid Live Editor：

https://mermaid.live/

它能导出：

- ASCII
- SVG
- PNG
- Markdown

---

# ⭐ 2. PlantUML（专业级）

特点：

- 语法强大
- 可画类图、时序图、组件图、部署图
- 可导出 ASCII
- 可集成 VSCode

示例：

```
@startuml
Frontend -> userPresencesStatus : send status
Frontend -> userNotViewingMarks : send beacon
userPresencesStatus --> MergeLogic
userNotViewingMarks --> MergeLogic
@enduml
```

# ⭐ 3. Graphviz（dot 语言）

- 更偏图论
- 可导出 ASCII（需要额外工具）
