## 详情说明

### 成员
- 成员页
- 成员详情页
```
成员页（成员列表页）、显示全部成员、可控制升序或降序排序
页面样式类似文件选择页，即双列布局，还有一点区别是不需要底栏

成员详情页，类似文件详情页，不过样式布局应改改
```

### 动态
- 动态页
```
显示在线成员

显示播放列表
- 全局所有正在播放的播放列表
- 热门的几个播放列表
- 最新的几个播放列表
- 附带导航，可跳转至播放列表页
```

动态相关的集合
- 动态用户状态集合
- 动态播放状态集合
```
动态用户状态集合
用于实现用户状态：
- 在线状态、在线但屏幕未显示状态、输入状态、离线状态
还附带用户当前的一些信息
- 用户当前正在播放的播放列表（关系字段）

动态播放状态集合
用于实现同步播放
用于获取正在播放的播放列表
用于获取热门播放列表（即查询前500条，出现多的即是热门的）
```
